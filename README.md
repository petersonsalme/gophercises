# Gophercises

This repository contains solutions to various Go exercises.

## Exercises Completed and Concepts Learned

* **Exercise 01: [Quiz](./quiz)**
  * **Concepts:** Reading and parsing CSV files, using the `flag` package for CLI arguments, and implementing concurrency with channels and `time.Timer` for a quiz timeout.
* **Exercise 02: [URL Shortener](./url-shortener)**
  * **Concepts:** Building an HTTP server with `net/http`, creating custom HTTP handlers, and parsing YAML files.
* **Exercise 03: [Create Your Own Adventure](./cyoa)**
  * **Concepts:** Parsing JSON files, working with `html/template` to render pages, interfaces, and implementing the Functional Options pattern for friendly APIs.
* **Exercise 04: [HTML Link Parser](./html-link-parser)**
  * **Concepts:** Parsing HTML using `golang.org/x/net/html`, depth-first search (DFS) over node trees, and writing unit tests in Go.
* **Exercise 05: [Sitemap Builder](./sitemap)**
  * **Concepts:** Web crawling, breadth-first search (BFS) for graph traversal, HTTP GET requests, and generating XML data.
* **Exercise 06: [Hacker Rank Challenges](./hacker-rank)**
  * **Concepts:** String manipulation and basic algorithms (solving CamelCase and Caesar Cipher challenges).
